# 🎵 **Music Unlocked**
*Listen Freely. Discover Without Limits.*

## 📖 About The Project (English)
Music Unlocked is an Android application built on the engine of NewPipe Extractor It allows you to stream music directly from YouTube for free, with zero ads, and complete user privacy.

## The Motivation
I live in the Republic of Latvia (EU). Due to licensing restrictions, I cannot access many popular tracks on SoundCloud without using a VPN. Paying for multiple streaming subscriptions isn't an option for me. That is why I decided to restore justice to our world of music. I fully understand that similar projects already exist, but I am doing this for my own experience.

## 🚧 Planned Features
🔊 **Ad-Free Streaming:** Enjoy your favorite tracks without interruptions.

📚 **Local Database:** User profiles, playlists, and track management (NewPipe integration) stored locally.

🎨 **Modern UI**: A clean, beautiful, and intuitive interface (inspired by Spotify's design).

📥 **Import Functionality**: Easily transfer your playlists and tracks from other music services.

This project is my way of restoring balance. It is built for personal use and as a learning experience, not for commercial distribution.

## 👨‍💻 Current Status

The app is currently under development. At this point, features for playing files from the database have been implemented. The database is stored on the local device, but a server can be set up in the future.
